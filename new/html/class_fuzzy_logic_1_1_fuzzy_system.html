Detailed Description

Main interface to the fuzzy logic system. Responsible for taking FuzzyLogic.CrispInput and returning FuzzyLogic.CrispOutput. Controls the FuzzyLogic.Fuzzifier, FuzzyLogic.InferenceEngine, and FuzzyLogic.Defuzzifier. Must be provided with FuzzyLogic.FunctionCurve and FuzzyLogic.FuzzyRulesList
